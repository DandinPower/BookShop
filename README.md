# TeamProject

# 使用說明

1. React前端
    - 進入Client/bookstore/的目錄開啟Terminal
2. Express後端
    - 進入Server/的目錄開啟Terminal

# 安裝方法

1. React前端
    
    ```bash
    npm install
    ```
    
2. Express後端
    
    ```bash
    npm install
    ```
    

# 運行伺服器

1. React前端
    
    ```bash
    npm start
    ```
    
2. Express後端
    
    ```bash
    npm run server
    ```
